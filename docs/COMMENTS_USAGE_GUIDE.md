## Common Questions

### Q: Can I delete someone else's comment?
**A:** No, you can only delete your own comments. Workspace owners may have this ability in future updates.

### Q: How many emojis can I add?
**A:** You can add one of each emoji type. Click again to remove.

### Q: Is there a comment limit?
**A:** Currently loads the last 100 comments. Pagination coming soon!

### Q: Can viewers comment?
**A:** Yes! All workspace members (Owner, Editor, Viewer) can comment and react.

### Q: Do comments save if I leave?
**A:** Yes! Comments are permanently saved to the database.

### Q: Can I format text (bold, italic)?
**A:** Not yet, but Markdown support is planned for future updates.
